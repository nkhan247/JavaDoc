### IfElse

```java

package com.nazmul.JavaDoc;

public class IfElse {

	public static void main(String[] args) {
		int age=19;
		if(age>18) {
			System.out.println("age greater than 18");
		}
		else {
			System.out.println("False");
		}

	}

}

```
